# authorization.py
Система авторизации на Flask с использованием SQLite3 и проверкой сложности паролей.

## Возможности
- Регистрация и вход пользователей
- Анализ сложности пароля (библиотека zxcvbn)
- Хеширование паролей (безопасное хранение)
- Ограничение длины логина и пароля (21 символ)
- Постоянные сессии (7 дней) и защита маршрутов

## Файл данных
Все данные пользователей хранятся в `users.db` (создаётся автоматически через SQLite3).
